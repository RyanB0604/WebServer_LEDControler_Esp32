Using my esp32, I uploaded index.html, ledFunctionality.js and styles.css to the ESP32. I needed to use littlefs to be able to manage and upload files, as well as creating the server, all of this is stored in the data folder. The esp32 connects to your local network and finds an open port, this port becomes the "domain" of the website. ESPAsyncWebServer and AsyncTCP work together to handle requests made by the Java Script file. The esp32 listens for these requests (e.g. "/toggleRedLED") and then executes the associated block of code. Currently you can toggle 3 different LEDs and also a Xmas mode which makes a LED lightshow! To use this code you must first upload the code to your esp32 and upload Filesystem Image using platformio, I used VS Code to do this. Then open the serial monitor on band 115200 and copy the IP address, this is how you will access the server. Make sure to match your pins on your breadboard with the global LED variables. Always make sure that the voltage is lowered through a resistor so you do not burn out your LEDs. Enjoy!
